# HR-DASHBOARD
The HR Analytics Dashboard is an interactive Power BI report designed to analyze employee data, track attrition trends, and provide insights into workforce demographics and satisfaction levels.
The image represents an HR Analytics Dashboard built in Power BI to visualize employee data and analyze attrition trends across different dimensions.

At the top, the dashboard displays key performance indicators (KPIs), including Total Employees (1400), Attrition Count (725), Attrition Rate (0.03%), Active Employees (2), and Average Age (39). These metrics provide a quick overview of the organization’s workforce status.
